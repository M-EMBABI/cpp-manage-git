# cpp-manage-git

This application will help you to add your files to git 
then commit with your message 
then push it to your repo


---

#### working on Linux only #

---
### compile software 
```
make 

```

### clean 

```

make clean 

```

## Running

```
  pusher_git <repo name>
  
```
